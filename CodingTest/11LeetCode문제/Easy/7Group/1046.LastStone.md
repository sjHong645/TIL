[문제](https://leetcode.com/problems/last-stone-weight/description/)

## 정답 코드 
``` java
class Solution {

    // stones[i] = i번째 스톤의 무게 
    public int lastStoneWeight(int[] A) {
    
        // () 안에 있는 기준을 세우면 
        // pq에 있는 큰 값이 먼저 poll하는 우선순위 큐가 생성된다. 
        PriorityQueue<Integer> pq = new PriorityQueue<>((a, b)-> b - a);
        
        
        // 배열 A에 값을 넣고
        for (int a : A) pq.offer(a);

        // pq의 사이즈가 1보다 큰 상황에서 
        // 가장 큰 두 숫자를 뺀 값을 넣는다. 
        while (pq.size() > 1) {
            pq.offer(pq.poll() - pq.poll());
        }
        
        return pq.poll();
    }
}
```
