## 정답 코드 
``` java
class Solution {
    public int[] arrayRankTransform(int[] arr) {
    
        // arr과 똑같은 배열 A를 만들고 오름차순 정렬한다. 
        int[] A = Arrays.copyOf(arr, arr.length);
        Arrays.sort(A);
        
        // key - arr에 있는 값
        // valeu = 순위 값 
        HashMap<Integer, Integer> rank = new HashMap<>();
        
        // A에 있는 값을 가지고 rank에 매핑한다. 
        for (int x : A) rank.putIfAbsent(x, rank.size() + 1);
        
        // rank.get(arr[i]) = arr[i] 값의 rank값 
        // 그걸 A에 넣고 return하면서 코드를 마무리한다. 
        for (int i = 0; i < arr.length; ++i) A[i] = rank.get(arr[i]);
        
        return A;
    }
}
```
