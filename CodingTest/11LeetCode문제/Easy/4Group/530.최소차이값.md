[문제](https://leetcode.com/problems/minimum-absolute-difference-in-bst/description/)

## 정답 코드 

``` java
    int minDiff = Integer.MAX_VALUE; // 트리의 전체에서 '최소값'을 찾아야 하기 때문에 전역변수로 선언했다. 
    TreeNode prev;
    
    public int getMinimumDifference(TreeNode root) {
        inorder(root); // root를 기준으로 inorder를 실행한다. 
        return minDiff;
    }
    
    public void inorder(TreeNode root) {
        if (root == null) return;
        
        // root의 왼쪽 
        inorder(root.left);
        
        // 원하는 동작
        // 전역변수 prev와 현재 노드 root의 값의 차이값을 minDiff와 비교하면서
        // 최소값을 찾아나간다. 
        
        // 이때, prev는 root가 되면서 계속해서 가리키는 노드를 바꿔나간다. 
        if (prev != null) minDiff = Math.min(minDiff, root.val - prev.val);
        prev = root;
        
        // root의 오른쪽 
        inorder(root.right);
    }
```
