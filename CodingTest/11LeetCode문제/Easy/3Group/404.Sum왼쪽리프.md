[문제](https://leetcode.com/problems/sum-of-left-leaves/description/)

## 내 코드 
``` java
class Solution {
    // 왼쪽 리프 노드의 합을 구해라
    public int sumOfLeftLeaves(TreeNode root) {

        if(root == null) return 0; 
        
        int sum = 0; 

        // 내 코드의 문제점
        // root의 왼쪽 서브트리의 오른쪽 서브트리가 존재할 수 있는데 
        // 그걸 무시한다. 
        sum += getLeftSum(root.left); 

        return sum + sumOfLeftLeaves(root.right);
    }

    private int getLeftSum(TreeNode node) {
        
        if(node == null) return 0; 

        while(node.left != null) {
            node = node.left; 
        }

        return node.val;

    }
}
```
## 정답 코드 
``` java
class Solution{
    public int sumOfLeftLeaves(TreeNode root) {
        if(root == null) return 0;
        int ans = 0;
        
        // 왼쪽 서브트리가 존재한다면 
        if(root.left != null) {
            
            // 그 왼쪽 노드가 리프 노드라면 ans의 값을 더해준다. 
            if(root.left.left == null && root.left.right == null) ans += root.left.val;
            
            // 그렇지 않다면 왼쪽 노드에 대해서 함수를 반복
            else ans += sumOfLeftLeaves(root.left);
        }
        
        // 오른쪽 서브트리에 대해서 함수를 반복한다. 
        ans += sumOfLeftLeaves(root.right);
        
        return ans;
    }
}
```
