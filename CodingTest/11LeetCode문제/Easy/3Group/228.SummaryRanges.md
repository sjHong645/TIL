[문제](https://leetcode.com/problems/summary-ranges/description/)

## 주요 내용

ex) nums = [0,1,2,4,5,7] 
```
i = 0 

a = nums[0] = 0

- (0+1) = 1 < nums.length && nums[1] - nums[0] == 1 => i값 1증가 => i = 1
- (1+1) = 2 < nums.length && nums[2] - nums[1] == 1 => i값 1증가 => i = 2
- (2+1) = 3 < nums.length && nums[3] - nums[2] = 2 != 1  while문 멈춤 

nums[2] = 2 != a이니까 "0->2"를 저장 
==============================
i = 3 

a = nums[3] = 4 
- (3+1) = 4 < nums.length && nums[4] - nums[3] == 1 => i값 1증가 => i = 4
- (4+1) = 5 < nums.length && nums[5] - nums[4] = 2 != 1  while문 멈춤 

nums[4] = 5 != a 이니까 "4->5"를 저장 
==============================
i = 5 

a = nums[5] = 7 

- (5+1) == nums.length while문 멈춤

nums[5] == a이니까 "7" 저장 

return "0->2","4->5","7"
```


## 정답 코드 
``` java
class Solution {
    public List<String> summaryRanges(int[] nums) {

        List<String> list = new ArrayList();
	
        for(int i=0;i<nums.length;i++){
        
            int a=nums[i]; // i번째 값 a 
            
            // i+1이 범위를 넘어서지 않으면서 
            // i+1번째 값과 i번째 값의 차이가 연속적일 때 
            // i값을 1씩 증가시킨다. 
            
            // 값이 연속적이지 않다면 while문을 멈춘다. 
            while(i+1 < nums.length && (nums[i+1]-nums[i])==1){
                i++;
            }
            
            // i번째 값이 
            // 처음에 설정한 a와 다른 경우 
            if(a!=nums[i]) list.add(a+"->"+nums[i]);
            
            // 똑같은 경우 
            else list.add(a+"");
        }
        return list;
            
    }
}
```
