[문제](https://leetcode.com/problems/is-subsequence/description/)

## 주요 내용 

`투 포인터`를 이용한다.

## 정답 코드 
``` java
public class Solution {
    public boolean isSubsequence(String s, String t) {
        if (s.length() == 0) return true;
        
        // s의 인덱스 indexS
        // t의 인덱스 indexT
        int indexS = 0, indexT = 0;
        
        // t의 인덱스가 범위를 넘지 않는 한 계속 반복 
        while (indexT < t.length()) {
            
            // 순서도 같아야 하기 때문에 아래와 같은 코드를 작성해야 한다. 
            
            // 문자열 t의 indexT번째 값과 
            // 문자열 s의 indexS번째 값이 동일하다면 
            
            // indexS을 하나 증가시킨다. 
            // 왜냐하면, 문자열 s가 부분문자열의 기준이기 때문이다. 
            
            
            if (t.charAt(indexT) == s.charAt(indexS)) {
                indexS++;
                
                // 그렇게 해서 indexS가 문자열 s 전체를 다 탐색했다면 true를 return한다.
                if (indexS == s.length()) return true;
            }
            
            indexT++;
        }
        
        return false;
    }
}
```
