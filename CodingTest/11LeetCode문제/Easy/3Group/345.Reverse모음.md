[문제](https://leetcode.com/problems/reverse-vowels-of-a-string/description/)

## 정답 코드 

``` java
public String reverseVowels(String s) {
    if(s == null || s.length()==0) return s;
    
    // 모음을 찾을 때 `if문`을 연속적으로 사용하지 않고 
    // 아예 `모음을 모아놓은 문자열`을 만들어서 `contains`로 알아낼 수 있다.
    String vowels = "aeiouAEIOU";
    char[] chars = s.toCharArray();
    
    int start = 0;
    int end = s.length()-1;
    
    // 범위를 넘어서지 않는 한 계속 반복한다. 
    while(start<end){
         
        // 범위를 넘지 않으면서 chars[start]가 모음이 아니면 한 칸 씩 이동한다.
        while(start<end && !vowels.contains(chars[start]+"")){
            start++;
        }
        
        // 범위를 넘지 않으면서 chars[end]가 모음이 아니면 한 칸 씩 이동한다.
        while(start<end && !vowels.contains(chars[end]+"")){
            end--;
        }
        
        // start와 end를 서로 바꾼다. 
        char temp = chars[start];
        chars[start] = chars[end];
        chars[end] = temp;
        
        // 이 부분을 쓰지 않아서 틀렸다. 
        // start 번째 문자와 end번째 문자를 서로 바꾸고 나서 
        // start와 end를 각각 한 칸씩 이동시켜줘야 한다.
        
        // 왜냐면, 그들은 이미 교환이 이뤄졌기 때문이다. 
        start++;
        end--;
    }
    return new String(chars);
}
```
