[문제](https://leetcode.com/problems/valid-perfect-square/description/)

## 주요 내용 

주어진 수가 `완전제곱수`인지 파악하는 방법을 `이진 탐색`으로 접근해서 풀어낼 수 있다. 

이때, `overflow`를 피하기 위해서 조금은 다른 방식으로 범위를 줄여나갔다. 

## 정답 코드 
``` java
class Solution {
    public boolean isPerfectSquare(int num) {
        int left = 1, right = num;
        
        while (left <= right) {
            
            // overflow를 피하기 위한 인덱스 중앙값 계산
            int mid = left + (right - left) / 2; // to avoid overflow incase (left+right)>2147483647
            
            // res = num을 mid로 나눈 몫
            // remain = num을 mid로 나눈 나머지 
            int res = num / mid, remain = num % mid;
            
            // num을 mid로 나눈 값이 mid랑 똑같고 나머지가 0이다는 뜻은
            // => num / mid = mid => num = mid * mid 을 의미한다. 
            // 때문에 이 조건을 만족한다면 true를 return한다.
            if (res == mid && remain == 0) return true; // check if mid * mid == num
            
            // ex) 36 / 3 = 12 => res = 12, mid = 3 
            // mid가 더 작으니까 mid의 값이 더 커지도록 범위를 큰 쪽으로 좁혀야 한다.
            
            // ex) 36 / 9 = 4 => res = 4, mid = 9
            // mid가 더 크니까 mid의 값이 더 작아지도록 범위를 작은 쪽으로 좁혀야 한다. 
            
            // 몫이 mid보다 클 때 == num을 지금의 mid보다 더 큰 값으로 나눠야 한다. 
            // 즉, 범위를 오른쪽으로 좁혀야 한다.
            if (res > mid) { // mid is small -> go right to increase mid
                left = mid + 1;
            } 
            
            // 몫이 mid보다 작을 때 == num을 지금의 mid보다 더 작은 값으로 나눠야 한다. 
            // 즉, 범위를 왼쪽으로 좁혀야 한다.
            else {
                right = mid - 1; // mid is large -> to left to decrease mid
            }
        }
        return false;
    }
}
```
