[문제](https://leetcode.com/problems/convert-sorted-array-to-binary-search-tree/)

## 주요 내용

`오름차순 정렬`된 배열을 `이진 탐색트리(BST)`로 변환해야 한다.

이진 탐색 트리의 성질 
1) 루트 노드의 값보다 `작은 값`은 루트의 왼쪽 서브 트리에 위치
2) 루트 노드의 값보다 `큰 값`은 루트의 오른쪽 서브 트리에 위치

때문에 아래 코드처럼 `중앙값`을 기준으로 

왼쪽 서브트리를 설정할 때는 `중앙값보다 작은` 값의 범위인 `low ~ (mid-1)` 에서  
오른쪽 서브트리를 설정할 때는 `중앙값보다 큰` 값의 범위인 `(mid+1) ~ high` 에서 설정했다. 

그리고 코드는 `이진탐색`코드와 흡사하다. 

## 정답 코드 

``` java
class Solution {

    public TreeNode sortedArrayToBST(int[] num) {
      if (num.length == 0) return null;
      
      TreeNode head = helper(num, 0, num.length - 1);
      return head;
    }

    public TreeNode helper(int[] num, int low, int high) {
        
        if (low > high) { // Done
            return null;
        }
        
        int mid = (low + high) / 2; // 중앙값이 위치한 index
        
        TreeNode node = new TreeNode(num[mid]);
        
        node.left = helper(num, low, mid - 1); // 중앙값보다 작은 값의 범위인 low ~ (mid-1)
        node.right = helper(num, mid + 1, high); // 중앙값보다 큰 값의 범위인 (mid+1) ~ high
        
        return node;
    }
}
```
