## 주요 내용 

단순히 `값`이 같은게 아니라 `같은 노드`를 가리켜야 한다. 즉, `메모리 주소`까지 서로 같아야 한다.

밑에있는 그림을 보면 이해가 쉽다. 

## 정답 코드 
``` java
public ListNode getIntersectionNode(ListNode headA, ListNode headB) {

    // boundary check
    if(headA == null || headB == null) return null;
    
    ListNode a = headA;
    ListNode b = headB;
    
    // if a & b have different length, then we will stop the loop after second iteration
    while(a != b){ 
    	  // for the end of first iteration, we just reset the pointer to the head of another linkedlist
        
        // a가 null이면 headB에서 다시 시작. 그렇지 않으면 next로 넘어감
        // b가 null이면 headA에서 다시 시작. 그렇지 않으면 next로 넘어감
        
        // 만약에 교차점이 없다면 a와 b가 둘 다 null이 되면서 while문을 빠져나온다. 
        // 교차점이 있다면 a == b가 되니까 while문을 빠져나온다. 
        a = a == null? headB : a.next;
        b = b == null? headA : b.next;    
    }
    
    return a;
}
```
